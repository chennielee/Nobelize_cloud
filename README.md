# Nobelize
**AI 기반의 글 평가/수정 서비스, “**Nobelize**”**  
- “Nobel(노벨 문학상) + Realize(실현하다)”  
- AI와 Docker/Kubernetes 클라우드 시스템을 활용한 글 평가/수정 서비스.




#
### **1. Python 가상환경**
1. 가상환경 생성

   ```bash
   python -m venv venv
   ```

2. 가상환경 활성화
   - Windows (CMD):
     ```bash
     venv\Scripts\activate
     ```

   - Windows (PowerShell):
     ```powershell
     .\venv\Scripts\activate
     ```

   - Linux/MacOS:
     ```bash
     source venv/bin/activate
     ```

3. 가상환경 비활성화
   ```bash
   deactivate
   ```

### **2. `requirements.txt`**

pip install -r requirements.txt
